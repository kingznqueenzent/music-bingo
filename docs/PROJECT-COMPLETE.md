## Quick checklist (verify everything works)

1. **Open the app**  
   Go to: https://music-bingo-kingzandqueenzentertainment-1662s-projects.vercel.app

2. **Host a game**  
   Click **Host a Game** → create with YouTube links or from Media Library → note the **game code**.

3. **Share with family**  
   Send them:  
   `https://music-bingo-kingzandqueenzentertainment-1662s-projects.vercel.app/join?code=GAMECODE`  
   (replace `GAMECODE` with your code). Or share the QR code from the host dashboard.

4. **Play**  
   They open the link → enter name → get their card. You play songs from the host dashboard; their card updates in real time.

5. **Optional**  
   - **Stage View:** Open the “Stage View” link in the host dashboard for a separate window (e.g. second screen).  
   - **Themes:** Use **Playlists** to host from a theme if you’ve seeded themes in Supabase.
